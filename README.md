# Self-Driving Vehicle UI Prototype

This workspace contains a touchscreen-focused UI prototype for a self-driving vehicle dashboard.

## 🚀 Run locally
1. Open `index.html` in your browser (recommended: Chrome, Edge, or Safari).
2. Use the on-screen controls to simulate vehicle status and interactions.

## 🧭 Features
- Navigation map placeholder with route controls
- Real-time status indicators (speed, range, traffic)
- Emergency stop & drive mode toggle
- Sensor health and diagnostics action
- Climate & comfort controls
- Media playback controls
- Passenger, security, and settings panels

## 📁 Files
- `index.html` — main UI layout
- `styles.css` — touch-friendly visual styles
- `app.js` — UI behavior and simulated vehicle state
